# GIS Maps

This folder contains the GIS engineering deliverables developed during the FTTH planning workflow.

The maps were created using QGIS and ArcMap and represent the spatial foundation of the network design.

## Map Deliverables

### High-Level Design (HLD)

Provides the strategic architecture of the network including:

* Feeder Fiber Routes
* FDH Locations
* Backbone Infrastructure
* Service Boundaries

### Low-Level Design (LLD)

Provides detailed deployment information including:

* Distribution Fiber Routes
* Duct Infrastructure
* FAT Locations
* NAP Locations
* Splice Closures

### Serviceability Analysis

Represents customer coverage and service readiness within the network.

## Purpose

These maps support:

* Telecom Engineering Design
* Infrastructure Planning
* Service Coverage Analysis
* Deployment Activities
* Asset Management
