# Bill of Quantities (BOQ)

This folder contains infrastructure quantity estimates generated during the FTTH planning process.

The BOQ was developed using GIS-derived measurements and engineering design outputs.

## Included Components

- Fiber Optic Cables
- Duct Infrastructure
- FDHs
- FATs
- NAPs
- Splice Closures
- Handholes
- Splitters
- Network Accessories

## Purpose

The BOQ supports:

- Cost Estimation
- Procurement Planning
- Resource Allocation
- Deployment Planning
- Infrastructure Management

The quantities presented within this folder were derived from the engineering design and GIS analysis workflow.
