# FTTH GIS Network Planning & Infrastructure Analytics

## Project Overview

This repository presents a complete Fiber To The Home (FTTH) planning and GIS infrastructure analytics project developed using telecom engineering principles, Geographic Information Systems (GIS), and business intelligence tools.

The project demonstrates the end-to-end workflow involved in designing, analyzing, and visualizing a modern GPON-based fiber optic network. It combines network planning, GIS mapping, serviceability analysis, Bill of Quantities (BOQ) preparation, optical network design, and Power BI dashboard development within a unified framework.

The project was developed as part of my professional GIS and telecom engineering portfolio and reflects practical industry workflows used in FTTH deployments.
